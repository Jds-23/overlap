# Overlap

A minimal timezone overlap tool. Compare working hours across timezones, find meeting windows, and pin specific times.

## Setup

```sh
pnpm install
pnpm dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `pnpm dev` | Start dev server |
| `pnpm build` | Typecheck + build |
| `pnpm lint` | Lint with ESLint |
| `pnpm test` | Unit tests (vitest) |
| `pnpm test:e2e` | E2E tests (playwright) |
