# Overlap

A minimal timezone overlap tool. Compare working hours across timezones, find meeting windows, and pin specific times.

## Setup

```sh
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Typecheck + build |
| `npm run lint` | Lint with ESLint |
| `npm run test` | Unit tests (vitest) |
| `npm run test:e2e` | E2E tests (playwright) |
